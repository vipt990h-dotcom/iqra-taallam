# خطوات نشر منصة اقرأ وتعلّم V3

1. فك الضغط عن الملف.
2. رفع محتويات المجلد إلى GitHub، وليس ملف ZIP نفسه.
3. التأكد من وجود index.html في جذر المستودع.
4. الذهاب إلى Settings ثم Pages.
5. اختيار Branch: main و Folder: /root.
6. فتح رابط GitHub Pages بعد دقيقة.
7. تجربة الصفحات والأيقونة والتسجيل من الجوال.

## قبل الإطلاق الرسمي
- ربط Firebase.
- إضافة حسابات المستخدمين.
- تجربة تسجيل الصوت.
- مراجعة الحقوق.
- استبدال البيانات التجريبية بالبيانات الحقيقية.
